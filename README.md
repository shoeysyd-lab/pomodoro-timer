# Pomodoro Timer

A simple, vanilla JavaScript Pomodoro timer web app.

## Features

- 25-minute work sessions
- 5-minute short breaks
- 15-minute long breaks after 4 work sessions
- Start, pause, and reset functionality
- PWA support for installation

## How to Use

1. Open `index.html` in a web browser
2. Click "Start" to begin a work session
3. The timer will automatically switch to breaks
4. After 4 work sessions, it suggests a long break

## Installation

This is a Progressive Web App (PWA). You can install it on your device by visiting the app in a supported browser and following the install prompt.

## Development

To run locally:

```bash
python3 -m http.server 8000
```

Then open http://localhost:8000 in your browser.
